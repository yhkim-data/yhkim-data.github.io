# yhkim-data.github.io

DI Lab (Data Insight Laboratory) 홈페이지

## 기술 스택

- Hugo Extended (v0.160.1) — 테마 없이 커스텀 레이아웃
- GitHub Pages 배포, GitHub Actions 자동 빌드
- 커스텀 도메인: [di-lab.io](https://di-lab.io)

## 로컬 개발

```bash
hugo server
```

`http://localhost:1313` 에서 확인

## 배포

`main` 브랜치에 push하면 GitHub Actions가 자동으로 빌드·배포합니다.
